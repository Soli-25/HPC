== Descrição ==

Tema personalizado desenvolvido especialmente para a Igreja House of Prayer Atlanta.
Design elegante em preto/branco/cinza com funcionalidade completa de blog integrada.

Recursos:
* Design responsivo e moderno
* Sistema de blog completo para o pastor
* Área de widgets no rodapé
* Menu de navegação personalizável
* Suporte a imagens destacadas
* Otimizado para SEO
* Compatível com Gutenberg
* Customizador do WordPress

== Instalação ==

1. No painel do WordPress, vá em: Aparência > Temas > Adicionar Novo
2. Clique em "Enviar Tema"
3. Escolha o arquivo hpc-atlanta.zip
4. Clique em "Instalar Agora"
5. Após a instalação, clique em "Ativar"

== Configuração Inicial ==

1. Vá em: Configurações > Leitura
   - Selecione "Uma página estática" para página inicial
   - Escolha sua página inicial personalizada
   - Escolha uma página para o blog

2. Vá em: Aparência > Menus
   - Crie um menu e adicione as páginas desejadas
   - Atribua ao local "Menu Principal"

3. Vá em: Aparência > Personalizar > Informações da Igreja
   - Configure endereço, telefone e email

== Como Publicar um Post ==

1. No painel do WordPress, vá em: Posts > Adicionar Novo
2. Digite o título do post
3. Escreva o conteúdo usando o editor Gutenberg
4. Adicione uma imagem destacada (recomendado: 1200x600px)
5. Escolha a categoria do post (Sermões, Estudos, etc.)
6. Clique em "Publicar"
